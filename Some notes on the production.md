# PT-BR
Para fazer esse programa eu me inspirei na vontade de querer jogos, mais especificamente o Naruto Shippuden: Ultimate Ninja 4 do PS2, localizados para minha língua (portugues brasileiro).

Pesquisando se tinha esse jogo em PT-BR, percebi que não tinha, apenas o começo, feito pelo @perrxd20 no X, no seguinte  <a href="https://x.com/perrxd20/status/1823032801970499967" target="_blank">post</a>. Como não tinham atualizações recentes dele, eu senti que poderia fazer algo pra isso ser mais fácil e rápido, visto que a principal dificuldade dele era ter que editar no <a href="https://mh-nexus.de/en/hxd/7" target="_blank">HxD</a>, que é bastante demorado e podendo corromper a ISO caso edite errado.

Daí, claro, usei AI pra fazer esse programa, o máximo que eu mesmo fiz foi trocar alguns valores de padding, posição e tamanho das janelas, mas praticamente tudo foi feito usando o ChatGPT (sei que não é o melhor pra isso, mas ajudou bastante). Fiquei em média 3-4 dias fazendo esse código, corrigindo os erros e ajustando para algo agradável e útil. Fiquei satisfeito com o resultado, e espero que você que vai usá-lo também fique :)

Para compilar ele em um executável, usei o <a href="https://tauri.app/" target="_blank">Tauri</a>, para não precisar usar diretamente o navegador para utilizar essa ferramenta, mas se quiser, vai estar disponível.

Fiz questão de salvar grande parte do processo de produção do código, divindo em pre-alpha, alpha, beta e release, foi realmente bastante coisa. Vou fazer o upload de cada um.

Também vou fazer upload da imagem de capa do projeto, fiz do zero no Photopea, me inspirando em icones da internet dos anos 2000 (pelo menos era pra ser), e refiz o simbolo do HxD.

o projeto teve alguns nomes, como:
* editor
* editor de texto com filtro e blocos
* HxD Clean Text Editor


# ENG
To make this program, I was inspired by the desire to have games, more specifically the PS2 game Naruto Shippuden: Ultimate Ninja 4, localized into my language (Brazilian Portuguese).

While searching to see if there was already a PT-BR version of the game, I realized there wasn’t one, except for the beginning of a translation made by @perrxd20 on X, in the following <a href="https://x.com/perrxd20/status/1823032801970499967" target="_blank">post</a>. Since there had been no recent updates from him, I felt I could create something to make the process easier and faster, considering that the main difficulty was having to edit everything directly in <a href="https://mh-nexus.de/en/hxd/7" target="_blank">HxD</a>, which is very time-consuming and can corrupt the ISO if edited incorrectly.

So, of course, I used AI to make this program. The most I personally did was adjust some padding values, positions, and window sizes, but practically everything was made using ChatGPT (I know it’s not the best tool for this kind of thing, but it helped a lot). I spent around 3–4 days working on this code, fixing bugs and adjusting things to make it pleasant and useful. I’m satisfied with the result, and I hope anyone who uses it will feel the same :)

To compile it into an executable, I used <a href="https://tauri.app/" target="_blank">Tauri</a>, so there would be no need to run the tool directly in a browser, although the browser version will also be available if desired.

I also made sure to save a large part of the development process, separating it into pre-alpha, alpha, beta, and release versions. There was actually a lot of progress over time, and I’ll upload each version.

I’ll also upload the project cover image. I made it from scratch in Photopea, inspired by internet icons from the 2000s (or at least that was the intention), and I also recreated the HxD symbol.

The project had a few names during development, such as:

* editor
* text editor with filter and blocks
* HxD Clean Text Editor
