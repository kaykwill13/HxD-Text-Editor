# ENG
To create this program, I was inspired by my desire to have games localized into my native language (Brazilian Portuguese), more specifically Naruto Shippuden: Ultimate Ninja 4 for the PS2.

While searching to see if this game had already been translated into PT-BR, I realized it hadn't—only the beginning of the game had been translated by @perrxd20 on X, in the following <a href="https://x.com/perrxd20/status/1823032801970499967" target="_blank">post</a>. Since there hadn't been any recent updates from him, I felt I could do something to make the process easier and faster, especially because his main challenge was having to edit everything directly in <a href="https://mh-nexus.de/en/hxd/7" target="_blank">HxD</a>, which is quite time-consuming and can corrupt the ISO if edited incorrectly.

Of course, I used AI to create this program. The most I personally did was adjust some padding values, positions, and window sizes, but practically everything was built with ChatGPT (I know it’s not the best tool for this kind of work, but it helped a lot). I spent around 3–4 days working on the code, fixing bugs and refining it into something useful and pleasant to use. I’m satisfied with the result, and I hope you will be too :)

To compile it into an executable, I used <a href="https://tauri.app/" target="_blank">Tauri</a>, so users wouldn't need to run it directly in a web browser. However, the browser version will also be available for those who prefer it.

(I made a point of saving a large portion of the development process, dividing it into pre-alpha, alpha, beta, and release versions. There was actually quite a lot of progress over time. I'll upload each version.) TEMPORARILY REMOVED, BUT I PLAN TO RE-UPLOAD THEM.

(I’ll also upload the project's cover image. I created it from scratch in Photopea, taking inspiration from internet icons from the 2000s—or at least that was the goal—and I also recreated the HxD symbol.) I HAVE SINCE REPLACED THE ICON AND LOGO WITH SOMETHING MORE ORIGINAL.

The project went through several names, including:

* Editor
* Text Editor with Filters and Blocks
* HxD Clean Text Editor
* HxD Text Editor
